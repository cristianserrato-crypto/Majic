# Majic

Majic es una landing page estatica en etapa inicial. Este repositorio queda separado para evolucionar el diseno, contenido y despliegue del sitio de forma independiente.

## Estructura

```text
index.html                  Pagina principal.
styles.css                  Estilos base.
nginx-site.example.conf     Referencia de publicacion con Nginx.
run-majic-static.sh         Servidor local simple para validacion.
```

## Ejecucion local

```bash
python3 -m http.server 8080
```
